# Maranto's Sewer & Water Services

Website for a local plumbing business serving Bloomingdale, IL.

Live site: [marantosws.com](https://www.marantosws.com)

## Running locally

Open `index.html` in a browser — no install or build step required.

## Tests

```bash
npm install
npm test
```
